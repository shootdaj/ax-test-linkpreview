## Running Tests

```bash
# Unit tests (fast, no dependencies)
npx vitest run --dir src

# Integration tests
npx vitest run --dir test/integration

# Scenario tests
npx vitest run --dir test/scenarios

# Full pyramid (run all in order)
npx vitest run --dir src && \
  npx vitest run --dir test/integration && \
  npx vitest run --dir test/scenarios
```
